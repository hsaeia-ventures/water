# PROYECTO: AppGTD - Ecosistema de Productividad Zen
**Descripción:** AppGTD es una aplicación web responsive diseñada bajo la metodología estricta de "Getting Things Done" (GTD) de David Allen. El objetivo central es permitir al usuario alcanzar el estado de "Mente como el agua" mediante la eliminación del estrés cognitivo.

**Puntos Clave del MVP:**
- **Filosofía:** Fricción cero en captura y rigor total en la organización.
- **Tecnología:** Construida con Tailwind CSS 4.1 (Engine Oxide), arquitectura Offline-First (IndexedDB + Service Workers) y una capa de IA invisible que actúa como asistente de pre-procesamiento y coach de ejecución.
- **Diseño:** Estética "Minimal-Robust". Uso de Progressive Disclosure para ocultar la complejidad hasta que sea necesaria.
- **Segmento Inicial:** Personal (escalable a Familiar y Empresa).
